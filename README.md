# Museo Virtual — Argentinos Juniors

Prototipo web navegable del museo histórico de AAAJ. HTML + CSS + Vanilla JS, sin build.

Hecho con datos públicos verificables (Wikipedia, sitio oficial) + placeholders explícitos donde se cargará la base de datos histórica del club.

## Stack

- HTML estático (sin build)
- Tailwind no — CSS custom en `styles.css`
- Vanilla JS con hash routing (`app.js`)
- Datos en `data.js`

## Para correr local

Doble click en `index.html` o servir con cualquier server estático (`python3 -m http.server 8080`).

## Deploy

Cualquier hosting estático: Vercel, Netlify, GitHub Pages.
